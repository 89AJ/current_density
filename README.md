# Current density calculations with gpaw

## Clone the repository
```
git clone https://github.com/89AJ/current_density.git
```

## Run a calculations
```
cd current_density/src/
```
```
python dump_ham.py
```
```
python calc_local.py
```
```
jmol c8/current.spt
```

## Requirements:
On Mac (& Linux?):

Install conda from website
```
pip install ase
```
```
pip install gpaw
```

On Windows:

Good luck!
